# 🔐 Cybersecurity Global Salaries 2026

Página interactiva con datos de salarios en ciberseguridad a nivel mundial, comparativa por país, roles, niveles y certificaciones clave.

## 📋 Contenido

- Rangos salariales anuales en USD por país
- Ranking global de roles mejor pagados
- Visualizaciones: CISO máximo, entry level, brecha mín–máx, distribución por demanda
- Metodología y fuentes

## 🚀 Uso

Abre `index.html` directamente en tu navegador o despliégala como GitHub Page.

## 🌐 Demo

[Ver página en vivo](https://0xvanguard.github.io/cybersecurity-global-salaries/)

## 📊 Datos

Los datos corresponden a rangos de compensación anual 2025–2026 en USD, basados en medianas ponderadas de múltiples fuentes públicas. Tipos de cambio de junio 2026.
